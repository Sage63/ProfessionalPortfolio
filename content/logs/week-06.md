**March 16** — Implemented the **Edit Profile** feature, enabling users to update personal information within the MediTrack mobile app.

**March 17** — Added **image avatar upload** capability for both mobile and web using Firebase Storage.

**March 18** — Integrated **push notification alerts** via Firebase Cloud Messaging to remind users about medication schedules.

**March 19** — Participated in the **Base Build Mapandan** community event.

> Push notifications are critical for a medication-tracking app — making sure they work reliably and don't miss a schedule truly matters for users.
